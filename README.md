### Project Overview
This project is a Java-based backgammon game. It features a graphical user interface and implements the classic rules of backgammon. 

### Building and Running
#### Prerequisites
- Java Development Kit (JDK).
- An Integrated Development Environment (IDE) like Vscode or Eclipse, or a basic text editor for manual compilation.

#### Compilation
1. Ensure JDK is installed and `javac` is available in your PATH.
2. Navigate to the project directory.
3. Compile all Java files:
   ```bash
   javac *.java
   ```

#### Running the Application
1. After compilation, run the game using the `BG` class (which contains the `main` method):
   ```bash
   java BG
   ```

### Notes
- The game is designed to be played on a single machine.
- Ensure all class files are in the same directory to avoid classpath issues.
- The game's GUI is built using Java's standard GUI toolkit.

### Potential Enhancements
- Network functionality for playing over a network.
- Enhanced AI for a more challenging single-player experience.
- Additional customization options for the game board and pieces.
- Saving and loading game states.
